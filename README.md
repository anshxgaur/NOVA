# 🚀 Quick Start

```bash
git clone https://github.com/anshxgaur/MODEL-X.git
cd MODEL-X

pip install -r requirements.txt
npm install

echo "GROQ_API_KEY=your_key" > .env
echo "VITE_GROQ_API_KEY=your_key" >> .env

python app.py
npm run dev
```
